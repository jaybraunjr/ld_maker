What it is

A lipid droplet is a phospholipid monolayer wrapped around a neutral-lipid core.

ld_maker builds that geometry as a ready-to-simulate structure. It reuses single-molecule conformations from your own equilibrated systems, writes a matching GROMACS topology, and hands off to the standard six-stage equilibration — no from-scratch solvation, no hand-editing coordinate files.

Two ways to build

Start from a bilayer, or from scratch.

recommended

Bilayer → LD

Take a solvated CHARMM-GUI bilayer, pry the two leaflets apart, and fill the midplane with TRIO. The bilayer's water and ions stay exactly where they are. Composition-agnostic — build cholesterol or a mixed phospholipid bilayer in CHARMM-GUI and it carries straight through.

from scratch

Grid build

Synthesize the whole monolayer / core / monolayer sandwich on a grid from single-molecule templates, with a rigid-body clash relaxation. Lipids and core only — solvate downstream. Handy for quick phospholipid-only variants.

Quick start

One command, start to finish.

Build the droplet, stage a self-contained run directory (coordinates, topology, index, protocol, force field), and minimize:

# insert 150 TRIO into a bilayer, stage everything, run minimization
bash ld_maker/pipeline.sh modules/move_prot/drude/bil/bil.gro 150 runs/ld1 --minimize

Then equilibrate — locally or on the cluster:

cd runs/ld1
bash run_equil.sh        # local (CPU)
sbatch run_equil.slurm    # CHPC GPU — finishes in under an hour

Prefer the library directly? ld_maker.bilayer_to_ld(bilayer, n_trio, out_gro, out_top) in Python, or python -m ld_maker --bilayer … on the command line.

The pipeline

Five stages from bilayer to droplet.

  1. 1

    Build

    Insert TRIO between the leaflets (or grid-build the trilayer). A loose initial core keeps the start clash-light; the box compresses later under pressure coupling.

  2. 2

    Stage

    Write topol.top (molecule counts in file order), an index.ndx with SOLU / MEMB / SOLV groups, and copy the CHARMM36 toppar and step6/step7 .mdp files.

  3. 3

    Minimize

    Steepest descent (step6.0) with position restraints on the lipid phosphates. Resolves the grid contacts before any dynamics.

  4. 4

    Equilibrate

    Six stages, step6.1 → step6.6: NVT then semi-isotropic NPT, with the lipid restraint ramped 400 → 0 kJ·mol⁻¹·nm⁻² as the core densifies.

  5. 5

    Produce

    Launch production (step7) from the equilibrated step6.6.gro.

Why the box holds its height early: the phosphate restraint is z-only (fx = fy = 0), so lipids diffuse laterally while the leaflets stay put. The core compresses only once that restraint is released over stages 6.4–6.6.

Running the dynamics

Local for setup, cluster for the run.

On CPU this system runs ~1 ns/day, so the full ~1.5 ns equilibration takes about a day and a half locally — fine for a smoke test, slow for real work. On a CHPC GPU node it finishes in under an hour. The generated run_equil.slurm targets swanson-gpu-np; set the GROMACS module name and submit.

Adding cholesterol or new lipids? Build the bilayer with them in CHARMM-GUI, then convert with bilayer mode. The insertion never inspects the leaflet composition, so sterols and mixed phospholipids just work — no packing code to trust.
